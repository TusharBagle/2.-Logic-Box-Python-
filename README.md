# Logic Box

## About
Logic Box is a collection of pattern programs created to improve programming logic and problem-solving skills. This project contains different types of patterns such as triangle patterns, star patterns, and number patterns.

## Patterns Included
- Right Angle Triangle Shape
- Reverse Right Angle Triangle Shape
- Left Angle Triangle Shape
- Reverse Left Angle Triangle Shape
- Pyramid Angle Shape
- Reverse Pyramid Angle Shape
- Diamond Shape

## Purpose
The main purpose of this project is to strengthen understanding of:
- Loops
- Conditions
- Nested loops
- Logic building
- Problem-solving techniques

- Python
